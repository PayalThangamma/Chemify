# Chemify

Chemify is an interactive periodic table learning web application designed to make chemistry concepts more engaging through visual interaction, quizzes, and feedback.

## Features

- Interactive periodic table interface
- Element-based learning experience
- Quiz-based feedback
- Sound-based interaction for engagement
- User-friendly frontend design

## Tech Stack

- React
- JavaScript
- HTML
- CSS

## Motivation

This project was built to explore how interactive web interfaces can improve learning engagement.  
It reflects my interest in user-facing applications and educational technology.

## Relevance

This project demonstrates my ability to build frontend interfaces using React, which is relevant for research tools that require interactive user interfaces.
